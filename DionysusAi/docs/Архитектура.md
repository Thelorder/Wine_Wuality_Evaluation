Архитектура на системата
Системата е изградена по клиент-сървър модел с ясно разделение между фронтенд и бекенд.
┌─────────────────────┐          ┌─────────────────────────────┐
│   Streamlit         │          │         FastAPI             │
│   Frontend          │◄────────►│   Backend API               │
│   (Python)          │  HTTP    │                             │
└─────────────────────┘          └─────────────┬───────────────┘
                                               │
                                     ┌─────────┴─────────┐
                                     │                   │
                           ┌─────────▼─────────┐   ┌─────▼─────┐
                           │  Random Forest    │   │   Ollama  │
                           │   Regression      │   │   (LLM)   │
                           │   Model (.pkl)    │   └───────────┘
                           └───────────────────┘
                                     ▲
                                     │
                           ┌─────────┴─────────┐
                           │     SQLite DB     │
                           │  (wine_samples)   │
                           └───────────────────┘
Основни компоненти:

Frontend – Streamlit приложение
Интерфейс за въвеждане на данни
Визуализация на резултати
Чат интерфейс
Симулация на сензори

Backend – FastAPI REST API
Обработка на заявки за предсказване
Съхранение и извличане на данни
Интеграция с ML модел
Комуникация с Ollama

ML компонент – RandomForestRegressor (scikit-learn)
LLM компонент – Ollama (локален модел, напр. llama3.1, phi3 и др.)
База данни – SQLite (еднофайлова)
